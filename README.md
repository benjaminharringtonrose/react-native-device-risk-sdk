# react-native-device-risk-sdk

React Native wrapper for Device Risk SDK

## Installation

```sh
npm install react-native-device-risk-sdk
```

## Usage

```js
import { multiply } from "react-native-device-risk-sdk";

// ...

const result = await multiply(3, 7);
```
